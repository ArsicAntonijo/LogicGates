# LogicGates
Simple logic gates simulator.
This app supports drawing following logic gates: and, or, nand, nor and not. Also there is supporting elements such as intput and output.

## Features and app logic
On right click pressed popup menu is opening, frow where you can choose which gate to draw or connect the gates.
For app to work properly it is important gates to be connect from right to left.
To connect two gates, you should first click connect from popup menu and than whith left click on mouse click the desired gate.
From main menu all gates can be removed and the color of gates and lines can be changed.
